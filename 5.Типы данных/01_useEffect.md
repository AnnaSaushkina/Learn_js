# useEffect

`useEffect` нужен для побочных эффектов — всего что происходит вне рендера компонента: запросы к серверу, подписки, таймеры.

В проекте запускается **один раз после первого рендера** — пустой массив `[]` гарантирует это. Идёт на сервер, получает задачи и кладёт их в стейт.

**Важно:** `useEffect` не сохраняет задачи — он их загружает. Сохранение через `saveTask`.

```ts
useEffect(() => {
  api.getTasks().then(setTasks);
}, []);
```
