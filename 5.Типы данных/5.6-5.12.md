## 1. Map и Set

1. Map — это объект, где можно чем угодно быть ключом
   - Ключами могут быть строки, числа, объекты, функции

2. Создание и базовые методы

```js
const map = new Map();

map.set(key, value); // записать
map.get(key); // прочитать (или undefined, если нет)
map.has(key); // проверить наличие ключа (true/false)
map.size; // количество пар ключ-значение
```

3. WeakMap принимает ключами только объекты и не мешает сборщику мусора удалить объект, если на него больше нет ссылок.

То есть метаданные “прикреплены” к объекту пользователя и не создают утечки памяти, когда пользовательский объект больше нигде не используется.

WakSet - коллекция которая может хранить только объекты
