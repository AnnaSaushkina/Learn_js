# stopPropagation и всплытие событий

События в DOM идут **снизу вверх** по дереву. Когда кликаешь на вложенный элемент — событие автоматически поднимается выше и срабатывают все обработчики на пути.

Кнопки «Удалить» и «Выполнено» вложены внутрь `div` с `onClick={() => onOpen(task)}`. Без `stopPropagation` клик на кнопку открывал бы карточку задачи.

```ts
<button onClick={(e) => {
  e.stopPropagation(); // останавливаем всплытие
  onDelete(task.id);
}}>
  Удалить
</button>
```

`stopPropagation` говорит браузеру — стоп, дальше не иди.
