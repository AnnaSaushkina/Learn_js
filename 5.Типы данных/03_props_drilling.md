# Props и Prop Drilling

Props — свойства компонента через которые передаются функции и данные. Без них компоненты не были бы интерактивными.

Передача через три уровня `AppLayout → TaskSection → TaskItem` называется **prop drilling**. `TaskSection` получает функции но сам их не использует — только передаёт дальше. Это лишний код.

**Zustand решает это** — любой компонент берёт нужное из стора напрямую:

```ts
// Вместо пробрасывания через все уровни:
const { deleteTask, toggleTask } = useTaskStore();
```
