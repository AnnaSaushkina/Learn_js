## Тип данных Symbol



 - **Symbol** — примитивный тип данных, представляющий уникальные идентификаторы
-  Создаются с помощью функции `Symbol()` или `Symbol.for()`
- Каждый символ уникален, даже если создан с одинаковым описанием
-  Символы используются:
   - Как ключи для скрытых свойств объектов
   - Для определения специальных системных свойств (well-known symbols)
   - Для избежания конфликтов имён свойств

---
- **Well-known symbols** — встроенные символы для настройки поведения объектов:
   - `Symbol.iterator` — для создания итераторов
   - `Symbol.toPrimitive` — для преобразования объекта в примитив
   - `Symbol.toStringTag` — для настройки вывода `Object.prototype.toString()`




##### Нужен для

1. создания уникальных идентификаторов и скрытых свойств
2. **Преобразование объектов** — важный механизм, управляемый тремя методами
3. **Symbol.toPrimitive** — способ контроля преобразований
4. для избегания неожиданного поведения объектов в операциях
